Pac-Man Game(C++)

Walk Warrior: Android Mobile Game App (Java, C#, and Unity )

  • Used Unity and C# to create the 2D-model for the main game scene including a main character, barriers, and collectables

  • Implemented a battle scene to fight with the monster. Used a Box Collider to switch between the main and the battle scene

  • Utilized the step sensor in Android to count the number of steps the user has took in the real world.
